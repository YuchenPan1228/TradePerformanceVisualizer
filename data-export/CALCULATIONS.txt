PORTFOLIO PERFORMANCE CALCULATIONS
===================================

DATA SOURCES:
-------------
1. Portfolio Data: Snaptrade API (connected Alpaca Paper account)
   - Current positions: get_user_account_positions()
   - Historical equity: get_reporting_custom_range() [WEEKLY data - 6 points over 30 days]
   - Daily reconstruction: Individual stock prices from yfinance (dividend-adjusted)

2. Benchmark Data: yfinance API
   - Market indices (SPY, etc.): Daily adjusted close prices
   - Adjusted Close includes: Stock splits + Dividends

3. Individual Stock Data: yfinance API
   - Daily prices with auto_adjust=True (dividend-adjusted)
   - Interval: 1d (daily)


PORTFOLIO VALUE CALCULATIONS:
-----------------------------

1. CURRENT PORTFOLIO VALUE (Dollar):
   Total Value = Σ (Current Price_i × Quantity_i) for all positions i
   
   Where:
   - Current Price_i = Latest market price from Snaptrade
   - Quantity_i = Number of shares held

2. DAILY PORTFOLIO RECONSTRUCTION:
   For each date d:
   Portfolio_Value_d = Σ (Price_d,i × Quantity_i) for all stocks i
   
   Where:
   - Price_d,i = Adjusted Close price of stock i on date d (from yfinance)
   - Quantity_i = Current quantity held (assumes no trading during period)
   - Note: This reconstructs daily values from individual stock prices

3. RETURN CALCULATION (Percentage):
   Return_d = ((Portfolio_Value_d - Portfolio_Value_0) / Portfolio_Value_0) × 100
   
   Where:
   - Portfolio_Value_0 = Portfolio value on first date (baseline)
   - Portfolio_Value_d = Portfolio value on date d

4. TOTAL RETURN:
   Total_Return = ((Current_Value - Cost_Basis) / Cost_Basis) × 100
   
   Where:
   - Cost_Basis = Σ (Average_Purchase_Price_i × Quantity_i)
   - Current_Value = Σ (Current_Price_i × Quantity_i)


BENCHMARK COMPARISON CALCULATIONS:
-----------------------------------

1. BENCHMARK NORMALIZATION (for Dollar Value comparison):
   When comparing portfolio to benchmark in dollar terms:
   
   Benchmark_Normalized_d = Benchmark_Price_d × (Portfolio_Value_0 / Benchmark_Price_0)
   
   This scales the benchmark to start at the same dollar value as the portfolio
   for meaningful comparison.

2. BENCHMARK RETURN CALCULATION:
   Benchmark_Return_d = ((Benchmark_Price_d - Benchmark_Price_0) / Benchmark_Price_0) × 100
   
   Where:
   - Benchmark_Price_0 = Benchmark price on first date
   - Benchmark_Price_d = Adjusted Close price on date d (includes dividends)

3. RELATIVE PERFORMANCE:
   Outperformance = Portfolio_Return_d - Benchmark_Return_d
   
   Positive = Portfolio outperforming benchmark
   Negative = Portfolio underperforming benchmark


DIVIDEND ADJUSTMENT:
-------------------
yfinance auto_adjust=True provides Adjusted Close prices that account for:
- Stock splits (price adjusted proportionally)
- Dividends (price adjusted downward by dividend amount on ex-dividend date)

Formula for Adjusted Close:
Adj_Close = Close × Split_Factor - Dividend_Adjustment

This ensures that total return calculations include dividend income.


ASSUMPTIONS & LIMITATIONS:
--------------------------
1. Daily reconstruction assumes:
   - No trading during the period (quantities remain constant)
   - All stocks are tradeable on yfinance
   - Prices are available for all trading days

2. Snaptrade weekly data:
   - Only provides 6 data points over 30 days (weekly frequency)
   - May not capture daily fluctuations
   - Daily reconstruction provides more granular data

3. Benchmark comparison:
   - SPY benchmark uses dividend-adjusted prices
   - Normalization assumes same starting value for fair comparison
   - Does not account for transaction costs or fees

4. Currency:
   - Portfolio values in CAD (from Snaptrade)
   - Benchmark (SPY) in USD
   - No currency conversion applied (assumes USD benchmark for comparison)


DATA INTERPOLATION:
-------------------
For missing dates (weekends, holidays), we use linear interpolation:
Value_d = Value_prev + (Value_next - Value_prev) × ((d - d_prev) / (d_next - d_prev))

Where:
- d = target date
- d_prev = previous available date
- d_next = next available date
